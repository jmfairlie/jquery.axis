# jquery.axis

A jQuery plugin to add graph axis like tickmarks to any container

## How To use
```
  $("#yourdiv").axis({start:0, length:100});

  //later on you can change the axis range by calling the update method

  $("#yourdiv").axis('update',[100,100]);
```
